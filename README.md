# CS50 Assignments (C)

This folder contains a few CS50 practice assignments written in **C**.

## Projects in this repository

- **if and else code**
- **sum of two numbers**
- **sum of two numbers with (scanf)**

## Notes

- Most projects are implemented as small standalone C programs (typically `main.c`).
- Build artifacts (e.g., `bin/`, `obj/`, `.o`, `.exe`) may be present in subfolders.

## How to run

If a project directory contains the compiled executable in `bin/`:

- Run the executable from that folder.

Otherwise, compile the program with a C compiler, for example (adjust filenames as needed):

```bash
gcc main.c -o main
./main
```
